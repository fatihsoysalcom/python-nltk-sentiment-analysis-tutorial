# Python NLTK Sentiment Analysis Tutorial

This Python script demonstrates how to perform sentiment analysis using NLTK. It preprocesses Twitter sample data, trains a Naive Bayes classifier, and predicts the sentiment of new text inputs.

## Language

`python`

## How to Run

1. Ensure you have Python 3 installed.
2. Install NLTK: `pip install nltk`
3. Run the script: `python sentiment_analysis.py`
4. The first time, it will prompt to download NLTK data. Uncomment the download lines if needed.
